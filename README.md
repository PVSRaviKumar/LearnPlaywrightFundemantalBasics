# Learn Playwright Fundamental Basics

This repository contains a basic Playwright test project to learn browser automation fundamentals.

## Project Structure

- `tests/` contains example Playwright test cases.
- `playwright.config.ts` configures the Playwright test runner.
- `package.json` lists the project dependencies.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the tests:
   ```bash
   npx playwright test
   ```

## Notes

- The project is configured for Chromium, Firefox, and WebKit.
- HTML test reports are generated in the `playwright-report/` folder.
